# 開発の目的
現代の社会では、デジタル化が進み紙媒体で物事の記録を行うということはなくなってきた。しかし、個人事業主などはシステムの導入にコストがかかりなかなか手が出ないこともある。そこで今回私は手軽に作れて費用も安く抑えられるようなアプリを作成した。
